OntoRama Demo

In order to use this application you will need to download Java Webstart .

Please Note: if your browser is Netscape 6, you will need to manually configure Netscape 6 for Java Webstart. To do this, go to Edit:Preferences Menu. In section Category, choose and expand section Navigator, click on 'Helper Applications' and if you don't have type for jnlp file already, click on 'New Type' button. When prompted, fill in following details: enter something like 'java webstart' for name, extension should be 'jnlp' and mime type 'jnlp'. Then click on 'Choose' button and select Java Webstart application (javaws). This will be done automatically in a later version of the Java Web Start

Once Webstart is installed - you are ready to launch OntoRama

If you have problems launching OntoRama

  1. make sure that Java Webstart is installed.
  2. try some other java webstart programs, for instance demos from sun's website: http://java.sun.com/products/javawebstart/demos.html

If your Java Webstart is installed correctly and when you try to run the above applications you get an error: "Unable to launch application", we suspect that your firewall may not be allowing files with extension .jnlp through (file needed to launch Webstart application). In this case try:

  • Reset your proxy setting in Java Webstart to manual. To do that: start Java Webstart, go to File:Preferences. In the dialog box choose 'General' tab. In 'Proxies' section click on 'Manual' and type in your settings.
  • If of the above fails - you can use an alternative distribution copy of OntoRama (see below) that is launched from local system rather then from the Web. We apologize for this inconvenience.

Running OntoRama locally

To run OntoRama locally, download ontorama.zip and unzip it on your local drive.

For instance, unzip into directory 'C:\ontorama' (Windows OS), then double click on OntoRamaLocal.jnlp.

If Java Webstart is installed correctly, you should see OntoRama loading.

If you wish to unzip ontorama into a different directory, you will need to edit OntoRamaLocal.jnlp file and specify alternative location by changing href in the followind line:

...
